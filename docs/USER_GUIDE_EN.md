## üöÄ 1. Getting Started

### Installation (PWA)
This app is a Progressive Web App (PWA). You can install it on your device for an offline, app-like experience.
- **Mobile (iOS/Android):** Tap "Share" (iOS) or the Menu (Android) and select **"Add to Home Screen"**.
- **Desktop (Chrome/Edge):** Click the install icon in the address bar.

### Layout Overview
- **Top Bar:** Quick toggles for **Language** (EN/FA) and **Theme** (Dark/Light). Settings icon.
- **Mode Switcher:** Tabs to switch between **Standard**, **Scientific**, **Currency**, **Time**, etc.
- **Display:** Shows input formula and result.
    - **Microphone Icon:** Click to use Voice Input (e.g., say "Two plus two").
    - **Tap to Copy:** Click the result text to copy it to your clipboard.
- **Keypad/Content:** The main interaction area changes based on the selected mode.

## üßÆ 2. Modes

### A. Standard Mode
Ideal for everyday arithmetic.
- **Operations:** Addition, Subtraction, Multiplication, Division.
- **Visual Math:** Uses symbols like `√ó` and `√∑` for better readability.
- **History:** Calculations are automatically saved.

### B. Scientific Mode
For advanced mathematical tasks.
- **Trigonometry:** `sin`, `cos`, `tan` (Support for **DEG** and **RAD** via the toggle on the display).
- **Functions:** Logarithms (`log`, `ln`), Roots (`‚àö`, `‚àõ`), Powers (`x¬≤`, `x‚Åø`).
- **Constants:** `œÄ` (Pi), `e` (Euler's number).

### C. Converter Mode
Convert between various units of measurement.
1. Select a **Category** (Length, Mass, Speed, etc.).
2. Enter the **Value** in the left box.
3. Select **From** and **To** units.
4. **Swap Button:** Quickly reverse the conversion.
5. **Save:** Click "Save to History" to keep a record of the conversion.

### D. Graphing Mode
Visualize mathematical functions using our high-performance native engine.
1. Enter a function in terms of `x` (e.g., `x^2`, `sin(x)`, `x + 5`).
2. The graph updates instantly.
3. **Pan:** Click and drag the graph area to move around.
4. **Zoom:** Use your mouse wheel or pinch gestures to zoom in and out.
5. **Reset:** Use the reset button to return to the center view.

### E. Equation Solver
Solve algebraic, geometric, and physics problems.
1. Select a tab (Linear, Quadratic, System, Pythagoras, etc.).
2. Enter the known values.
3. Click **Solve**. The result and the full equation are saved to History.

### F. Matrix Calculator (New v3.1)
Perform linear algebra operations.
1. **Setup:** Use the number inputs to define dimensions for Matrix A and Matrix B (up to 5x5).
2. **Input:** Type numbers into the grid cells.
3. **Operations:**
   - **A + B / A - B:** Adds or subtracts matrices.
   - **A √ó B:** Performs matrix multiplication (Columns of A must match Rows of B).
   - **Det(A) / Inv(A):** Calculates determinant or inverse of Matrix A.
   - **Trans(A):** Transposes Matrix A.

### G. Statistics (New v3.1)
Analyze datasets.
1. **Input:** Enter a list of numbers separated by commas or spaces in the text box (e.g., `10, 20, 5, 8.5`).
2. **Calculate:** Click the calculate button.
3. **Results:** View Mean, Median, Mode, Variance, Standard Deviation, etc.

### H. Currency & Time (v3.0)
- **Currency:** Real-time exchange rates. Select base currency and amount.
- **Time:** World clock and timezone comparison tool.
